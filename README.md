# Kingshot Rally Sync Timer — Kingdom 1564

本版本包含：

- 中英文同頁
- 2～5 重集結
- 行軍時間自動排序
- 自動計算開集結間隔
- 全螢幕大型倒數
- 自動切換下一位玩家
- 暫停、繼續、重新倒數
- 一鍵複製集結順序
- 手機版介面

## GitHub Pages 更新方式

將以下檔案上傳並覆蓋 Repository 根目錄原檔案：

- index.html
- style.css
- script.js

等待 GitHub Pages 重新部署後即可使用。
